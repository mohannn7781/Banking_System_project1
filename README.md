# Banking_System_project1
my 1st ever project on object oriented programing,this helped me in understanding the oops in real time


A simple banking system built using Java OOP concepts.

## Features
- Create customers
- Create savings/current accounts
- Deposit money
- Withdraw money
- Search customers
- Search accounts

## Concepts Used
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation
- ArrayList Collections
